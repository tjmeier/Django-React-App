# Starting the App
## Setting up the Repo
* Clone Repo
* `$ cd`

## Deployment
* Pull Changes
* `$ python manage.py makemigrations`

# Manual Backup
`$ python manage.py dumpdata -a -o data.json`

# Committing
* Encusre no sensitive or data files will be committed
* `$ pip freeze > requirements.txt`
